# autoG6Flow
基于antV G6中以html为节点的自动分层布局数据可视化展示
## 运行项目

```
 git clone https://github.com/wlc534/autoG6Flow

 cd autoG6Flow

 open index.html with Browser

 
```
## 说明

>  本项目主要是用antV 家族中的 G6  进行以html自定义节点的自动布局连线展示
